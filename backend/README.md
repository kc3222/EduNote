# EduNote Backend

FastAPI backend with JWT authentication.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the server:
   ```bash
   python3 -m uvicorn main:app --reload --port 8000
   ```
